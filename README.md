# 🔎 Adivina el número secreto:

Este repositorio contiene un juego de nivel básico de programación desarrollado en Python. 

## 🚀 Objetivo del programa

Crear un código sencillo de python que genere un número secreto y el usuario deba descubrirlo mediante pistas de "mayor" o "menor". 

## 📋 Requisitos de la Misión

* Generar un número aleatorio entre 1 y 100.
* Solicitar intentos al usuario de forma persistente.
* Validar que la entrada sea un número (evitando que el programa se rompa).
* Proporcionar pistas < o >.
* Contabilizar el número de intentos hasta el éxito.

## 🧠 Arquitectura del Código

Para resolver este reto, hemos estructurado el programa utilizando:

1. Bucles (while): Un bucle interno para la lógica de los intentos del usuario.
2. Condicionales (if, elif, else): Para comparar el número del usuario con el secreto.
